Basic Declaration

This application (Thanox) does not collect any of your personal information. It only queries the following non-personal information when necessary:

  •   Read the list of installed applications; only used when you configure applications, and this information is completely checked locally, with the results not being uploaded anywhere.
  •   Read the list of multiple users on the device; used for supporting multi-user configurations for certain functions, only reads the user ID, usually an Arabic numeral. This information is completely checked locally, with the results not being uploaded anywhere.
  •   Read the device's model, system version number, Android ID (automatically assigned by the system for each app, unrelated to the device); only used for advanced version eligibility verification in the PRC version, and it does not contain any personal or private information.

Requested Permissions

  •   Read and write external storage; used for data caching, data backup, and importing/exporting certain functions.
  •   Network; used for online advanced version eligibility verification.
  •   Install applications: used to guide users to manually install proxy applications for intelligent freezing with user consent.
  •   Uninstall applications: used to guide users to manually uninstall proxy applications for intelligent freezing with user consent.
  •   Read the list of installed applications; only used when you configure applications.
